Para clonar e rodar qualquer um desses projetos, siga estas etapas simples:

Clone o repositório:
bash

Copiar código

Navegue até o diretório do projeto desejado:
bash
Copiar código
cd projeto1

Compile o código (se necessário) e execute-o.
